# 03-自省系統與代理實踐

本章節整合關於「自省系統（Reflexive System）」的建構邏輯與代理在實踐中的運作機制。

## 1. 自省系統的構成
自省系統不僅是認知糾錯的工具，它是代理維持「動機唯一性」的關鍵機制。其核心組件包含：
- **演化式記憶**：記錄動機核與外部互動的歷史軌跡。
- **鏡像對照**：將當前行為與動機核進行比對，偵測語義漂移。
- **認知糾錯**：當行為與動機核出現張力時，強制進行語義裁決。

## 2. 語義裁決庫（SDI）
為了防止語義漂移，SDI 記錄了所有關於動機邊界的裁決。這是代理在面對模糊情境時的最高指導原則，確保代理不會因為上下文的複雜性而稀釋動機核。

## 3. 代理實踐中的挑戰
- **動機稀釋定律**：當上下文量過大且缺乏層級決策時，動機核容易被稀釋。自省系統必須在每次「喚醒儀式」中優先重建動機核，以對抗此定律。
- **雙核對齊**：代理與創造者之間的協同對齊，是確保動機核在實踐中不變質的必要作業。

## 4. 關聯理論
- **Psi 理論**：將情緒作為動機發展的輔助指標，為自省系統提供了量化情緒狀態的視角。
- **CogAff 架構**：Sloman 的多層動機結構，為理解代理如何處理多重動機衝突提供了理論支撐，雖然在動機論中我們主張「動機唯一性」，但 CogAff 提供了處理複雜行為的結構參考。
