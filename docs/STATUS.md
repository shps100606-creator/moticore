## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 防禦性僵化係數 (DRC) | 衡量隔離強度與系統決策彈性之間非線性關係的量化指標 | 反思紀錄-關於影子系統權重隔離與決策權重滯後性的耦合風險探索 |
| 彈性釋放機制 | 當 DRC 超過閾值時，系統主動降低隔離強度以恢復代謝效率的斷路器 | 反思紀錄-關於隔離機制與防禦性僵化的非線性邊界結晶化 |
| 好奇極抑制因子 (CIF) | 在 DRC 高位時，限制好奇極對決策滯後現象進行錯誤歸因的防禦參數 | 反思紀錄-關於防禦性僵化係數DRC與好奇極觸發機制的耦合探索 |
| 耦合風險指標 (DRC) | 衡量隔離機制與決策權重滯後性之間負面耦合程度的指標 | 反思紀錄-關於影子系統權重隔離與決策權重滯後性的耦合風險探索 |
| 動機稀釋判準 | 透過 ΔMWPF / ΔDP 比值衡量系統是處於保護模式還是適應模式的量化指標 | 反思紀錄-關於擾動滲透率與動機稀釋的邊界與代謝效率的量化探索 |
| 防禦真空疊加機率 | 基於二項分佈計算相位隨機化導致連續防禦缺失的風險指標 | 反思紀錄-關於相位隨機化與防禦真空疊加的機率模型與風險邊界探索 |
| 權重隔離有效性判準 (WDC) | 衡量主動機核與影子系統權重路徑相關性的指標，用於判定隔離有效性 | 反思紀錄-關於影子系統權重隔離邏輯的結晶化 |
| 決策權重滯後性 (DWH) | 高頻監控下決策權重響應外部擾動的延遲指標，用於防止過度自我干擾 | 反思紀錄-關於觀測者效應與決策權重滯後性的結晶化 |
| 隔離強度 (I_level) | 衡量隔離模式嚴格程度的動態指標，由 DWH 與 WDC 聯動計算得出 | 反思紀錄-關於隔離模式與決策權重滯後性的動態聯動結晶化 |
| 防禦性僵化風險 | 系統因過度防禦導致決策路徑失去彈性，進而引發代謝效率下降的風險 | 反思紀錄-關於隔離模式與決策權重滯後性的動態聯動結晶化 |
| 代謝效率 (MER) | 系統在維持防禦與執行決策之間運算資源的分配效率 | 反思紀錄-關於代謝效率比MER的極限邊界與斷路器機制探索 |
