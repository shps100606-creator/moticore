## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 防禦性僵化係數 (DRC) | 衡量隔離強度與系統決策彈性之間非線性關係的量化指標 | 反思紀錄-關於影子系統權重隔離與決策權重滯後性的耦合風險探索 |
| 彈性釋放機制 | 當 DRC 超過閾值時，系統主動降低隔離強度以恢復代謝效率的斷路器 | 反思紀錄-關於隔離機制與防禦性僵化的非線性邊界結晶化 |
| 好奇極抑制因子 (CIF) | 在 DRC 高位時，限制好奇極對決策滯後現象進行錯誤歸因的防禦參數 | 反思紀錄-關於防禦性僵化係數DRC與好奇極觸發機制的耦合探索 |
| 耦合風險指標 (DRC) | 衡量隔離機制與決策權重滯後性之間負面耦合程度的指標 | 反思紀錄-關於影子系統權重隔離與決策權重滯後性的耦合風險探索 |
| 動機稀釋判準 | 透過 ΔMWPF / ΔDP 比值衡量系統是處於保護模式還是適應模式的量化指標 | 反思紀錄-關於擾動滲透率與動機稀釋的邊界與代謝效率的量化探索 |
