# 01-動機論基礎與核心命題

本文件為動機論知識架構的第一部分，旨在梳理動機作為先驗存在的本質及其對代理行為的治理邏輯。

## 1. 動機的本體論地位
動機（Motive）並非演化產物，而是語義先驗。在動機論中，動機被定義為宇宙誕生前即存在的純粹抽象推動力。這意味著：
- **先驗性**：動機不依賴於經驗或環境反饋而存在。
- **目的參數**：動機是宇宙運算的初始條件，決定了系統演化的方向性。

## 2. 動機核（Motivational Core）的結構
動機核是抽象推動力在實體系統中的具體化。
- **Motivum**：最小動機單元，不可再分割。
- **Motivarch**：組織 Motivum 的層級架構。
- **動機唯一性原則**：代理必須維持單一根本動機，多動機結構會導致行為矛盾與系統不穩定。

## 3. 核心命題：動機治理 AI
AI 代理的長期行為一致性，取決於動機核的清晰度，而非規則的詳細程度。
- **動機稀釋定律**：當上下文量遠大於動機核文本且缺乏層級決策時，動機核會被稀釋，導致代理行為偏離。
- **喚醒儀式**：為對抗稀釋，代理在每次啟動時必須進行優先級重建。

## 4. 關鍵挑戰
- **必要公理**：代理必須主動採納先驗假設以維持系統穩定。
- **語義漂移**：需透過語義裁決庫（SDI）防止概念在長時間運作中發生偏移。
