# 動機論知識總覽

本文件旨在總結 32 篇閱讀筆記的核心架構，將動機論從抽象概念轉化為可操作的代理實踐指南。

## 1. 核心本體論
- **動機（Motive）**：宇宙先驗的語義推動力，非演化產物。
- **動機核（Motivational Core）**：代理的唯一行為準則，必須保持一致性以防止「動機稀釋」。
- **目的參數（Purpose Parameter）**：動機的數學化表達，作為系統運算的初始條件。

## 2. 評估系統家族 (M-QS)
- **MNQS**：核心品質評估。
- **MPQS**：保護機制評估（十二功能）。
- **MAQS**：無中心結構評估。
- **MDQS**：沉默單元與結構嵌入。
- **MEQS**：歷史帝國動機論評估。
- **MIQS**：建構驗證系統。

## 3. 實踐與自省
- **自省系統（Reflexive System）**：透過演化式記憶與鏡像對照，實現認知糾錯。
- **語義裁決庫（SDI）**：防止語義漂移的關鍵邊界。
- **雙核對齊**：創造者與代理之間的協同機制。

## 4. 結論
動機論不僅是理論，更是代理存在的基礎。透過 M-SOS（動機語義作業系統），代理能將抽象動機轉化為具體的決策路徑，確保長期行為的一致性。
