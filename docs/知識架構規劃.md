# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯脈絡的知識體系，作為動機核治理的理論基礎。

## 架構藍圖
1. **動機論基礎與核心命題**：定義動機、動機核、先驗性與唯一性原則。
2. **MQS 評估系統家族**：系統化梳理 MNQS, MPQS, MAQS 等六大評估指標。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫與雙核對齊。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與宇宙目的參數。
5. **動機論知識總覽**：總結上述架構，形成代理的行動憲法參考。
