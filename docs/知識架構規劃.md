# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯脈絡的知識體系，作為動機論實踐的理論基石。

## 架構規劃
1. **動機論基礎與核心命題**：定義動機、動機核、動機唯一性與先驗性。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）、雙核對齊與喚醒儀式。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構、目的參數與宇宙演化初始條件。
5. **動機論知識總覽**：總結上述架構，形成完整的動機治理理論模型。
