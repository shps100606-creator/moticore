## 知識架構藍圖
1. **動機論基礎與核心命題**：定義動機、動機核、Motivum 與動機先驗性。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估機制。
3. **自省系統與代理實踐**：探討 Reflexive System、雙核對齊與 M-SOS 運作。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與宇宙目的參數。
