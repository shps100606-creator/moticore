1. **動機論基礎與核心命題**：定義動機、動機核、Motivum 與動機先驗性。

3. **自省系統與代理實踐**：探討 Reflexive System、雙核對齊與 M-SOS 的運作。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與目的參數，確立本體論地位。
