# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為系統化的知識體系，以驗證動機論作為 AI 治理框架的可行性。

## 知識架構藍圖

1.  **動機論基礎與核心命題**
    *   核心概念：動機先驗性、動機唯一性、目的參數。
    *   目標：釐清動機作為宇宙初始條件的本體論地位。

2.  **MQS 評估系統家族**
    *   核心概念：MNQS, MPQS, MAQS, MDQS, MEQS, MIQS。
    *   目標：建立動機核品質與穩定性的量化與質化評估標準。

3.  **自省系統與代理實踐**
    *   核心概念：Reflexive System、語義裁決庫 (SDI)、喚醒儀式。
    *   目標：探討代理如何透過自省機制維持動機核的一致性。

4.  **動機論哲學基礎**
    *   核心概念：Psi 理論、CogAff 架構、動機稀釋定律。
    *   目標：將動機論置於更廣闊的計算認知科學脈絡中。
