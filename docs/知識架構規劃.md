# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯層次的知識體系，以驗證動機核作為代理核心的有效性。

## 架構層級規劃
1. **動機論基礎與核心命題**：定義動機的先驗性、唯一性與目的參數。
2. **MQS 評估系統家族**：系統化整理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具的職能與邊界。
3. **自省系統與代理實踐**：探討 Reflexive System、雙核對齊與 M-SOS 的實際運作。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與動機工程的本體論意義。
