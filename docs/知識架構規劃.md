1. **基礎與核心**：定義動機論的本體論基礎（Motivum, 動機核, 唯一性原則）。
2. **評估系統家族**：梳理 M-QS 家族（MNQS, MPQS, MAQS 等）的演進與功能。
3. **實踐與自省**：分析自省系統（Reflexive System）與代理實踐中的認知糾錯。
4. **哲學與總覽**：探討動機先驗性、目的參數與宇宙演化。
