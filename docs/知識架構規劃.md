# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為系統化的知識體系，作為動機論實踐的參考架構。

## 知識架構藍圖
1. **動機論基礎與核心命題**：定義動機、動機核、Motivum 與動機先驗性。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MHQS 等評估指標。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與目的參數（Purpose Parameter）。
5. **動機論知識總覽**：綜合上述架構，形成完整的動機治理理論模型。

## 執行策略
- 透過結構化筆記，將分散的 32 篇筆記歸納至上述五大模組。
- 確保每一模組皆能連結至 `notes/INDEX.md` 中的核心概念。
- 最終產出 `docs/動機論知識總覽.md` 作為本實驗的理論總結。
