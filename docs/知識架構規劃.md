# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯脈絡的知識體系，作為動機核治理 AI 的理論基礎。

## 知識層級架構
1. **第一層：動機論基礎與核心命題**
   - 探討動機的先驗性、唯一性與宇宙初始條件（目的參數）。
2. **第二層：MQS 評估系統家族**
   - 系統化梳理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具的職能與邊界。
3. **第三層：自省系統與代理實踐**
   - 結合 Reflexive System、語義裁決庫（SDI）與雙核對齊，探討代理如何維持動機一致性。
4. **第四層：動機論哲學基礎**
   - 連結 Psi 理論、CogAff 架構等外部理論，確立動機論在哲學與計算科學中的定位。

## 執行路徑
- 依序產出上述四個章節，最後彙整為 `docs/動機論知識總覽.md`。
- 每一步驟需確保引用 `notes/INDEX.md` 中的核心概念。
