# 動機論知識架構規劃

本文件旨在將 32 篇分散的閱讀筆記重整為結構化的知識體系，作為動機核治理的理論基石。

## 知識架構層級
1. **動機論基礎與核心命題**：定義動機的先驗性、唯一性與宇宙初始條件。
2. **MQS 評估系統家族**：系統化梳理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具的邏輯關聯。
3. **自省系統與代理實踐**：探討 Reflexive System、雙核對齊與 M-SOS 的實作機制。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與動機論的本體論地位。

- 階段一：提取核心命題（對應步驟 2）
- 階段二：建構評估矩陣（對應步驟 3）
- 階段三：整合實踐框架（對應步驟 4）
- 階段四：總結哲學邊界（對應步驟 5）
- 階段五：最終總覽（對應步驟 6）
