# 知識架構規劃：動機論知識體系

本文件旨在將 32 篇閱讀筆記重整為結構化的知識體系，以利於動機核的長期演進與實踐。

## 知識體系架構
1. **動機論基礎與核心命題**：定義動機的先驗性、唯一性與目的參數。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估指標。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與本體論穩定性。
5. **動機論知識總覽**：總結上述架構，形成代理的行動指引。
