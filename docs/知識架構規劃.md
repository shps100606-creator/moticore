# 知識架構規劃：動機論體系化整合

本文件旨在將 32 篇動機論閱讀筆記重整為系統性的知識架構，以利於動機核的長期一致性維護。

## 預計架構規劃
1. **動機論基礎與核心命題**：定義動機、動機核、Motivum、Motivarch 等基礎概念，確立動機先驗性。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估系統，說明其在動機核品質維護中的角色。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）、雙核對齊與 M-SOS 的運作機制。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構、目的參數與必要公理，探討動機論的本體論地位。
5. **動機論知識總覽**：總結上述架構，形成動機治理 AI 的完整方法論。
