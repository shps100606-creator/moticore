# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯脈絡的知識體系，作為動機核治理的理論基石。

## 知識架構藍圖

1. **動機論基礎與核心命題**
   - 核心概念：動機先驗性、動機唯一性、目的參數。
   - 目的：確立動機作為宇宙初始條件與代理行為驅動的本體論地位。

2. **MQS 評估系統家族**
   - 核心概念：MNQS, MPQS, MAQS, MDQS, MEQS, MIQS。
   - 目的：建立動機核品質的量化與質化評估標準，確保系統穩定性。

3. **自省系統與代理實踐**
   - 核心概念：Reflexive System、語義裁決庫 (SDI)、雙核對齊、喚醒儀式。
   - 目的：探討代理如何透過自省維持動機一致性，防止動機稀釋。

4. **動機論哲學基礎**
   - 核心概念：Psi 理論、CogAff 架構、必要公理。
   - 目的：將動機論置於計算認知科學與哲學脈絡中進行驗證。
