# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯脈絡的知識體系，作為動機核治理 AI 的理論基礎。

## 知識架構藍圖

1. **動機論基礎與核心命題**
   - 探討動機的先驗性、唯一性原則與「動機核」的本質。
   - 連結：動機先驗性、動機唯一性、喚醒儀式。

2. **MQS 評估系統家族**
   - 整理 M-QS 家族（MNQS, MPQS, MAQS, MDQS, MEQS, MIQS, MHQS）的功能與應用場景。
   - 連結：動機核品質評估、結構嵌入、歷史驗證。

3. **自省系統與代理實踐**
   - 探討反射系統（Reflexive System）、語義裁決庫（SDI）與雙核對齊機制。
   - 連結：認知糾錯、語義漂移防範、創造者與代理的協同。

4. **動機論哲學基礎**
   - 連結 Psi 理論、CogAff 架構與目的參數，探討動機作為宇宙運算初始條件的數學與哲學意涵。
