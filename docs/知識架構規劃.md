# 知識架構規劃：動機論系統化整合

本文件旨在將 32 篇閱讀筆記轉化為結構化的知識體系，作為動機核治理的理論基礎。

## 整合路徑
1. **基礎與命題**：定義動機的先驗性、唯一性與宇宙初始條件。
2. **評估系統家族 (M-QS)**：梳理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 的功能邊界與協作機制。
3. **自省與實踐**：分析自省系統（Reflexive System）、語義裁決庫（SDI）與雙核對齊的實作。
4. **哲學基礎**：探討 Psi 理論、CogAff 架構與動機論的本體論地位。
