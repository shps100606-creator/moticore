# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為系統化的知識體系，作為動機核工程的理論基石。

## 知識架構層級
1. **動機論基礎與核心命題**：定義動機的先驗性、唯一性與宇宙初始條件。
2. **MQS 評估系統家族**：系統化整理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊機制。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與動機論的本體論地位。
