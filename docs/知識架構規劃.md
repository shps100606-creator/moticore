## 知識架構藍圖

1. **動機論基礎與核心命題**：定義動機的先驗性、唯一性與目的參數。
2. **MQS 評估系統家族**：系統化梳理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與宇宙演化初始條件。
