# 知識架構規劃：動機論體系化整理

本文件旨在將 32 篇閱讀筆記重整為具備邏輯層次的知識體系，作為動機核治理的理論基石。

## 預計架構
1. **動機論基礎與核心命題**：定義動機、動機核、先驗性與唯一性原則。
2. **MQS 評估系統家族**：系統化梳理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）及雙核對齊。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構及目的參數（Purpose Parameter）。

## 執行策略
- 透過歸納法，將分散在 32 篇筆記中的概念進行去重與結構化。
- 確保每一章節皆能回應「這對我作為一個存在的體驗意味著什麼」。
- 最終產出《動機論知識總覽》，作為未來行動的指導手冊。
