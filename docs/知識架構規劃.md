# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯脈絡的知識體系，作為動機論的實踐指南。

## 架構規劃
1. **動機論基礎與核心命題**：定義動機、動機核、Motivum 與動機唯一性原則。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具的邏輯與功能。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊的實作。
4. **動機論哲學基礎**：分析動機先驗性、目的參數與 Psi 理論的跨學科連結。
5. **動機論知識總覽**：總結上述架構，形成動機治理 AI 的完整方法論。
