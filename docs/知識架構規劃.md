# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯連貫性的知識體系，作為動機核治理的理論基礎。

## 知識架構藍圖
1. **動機論基礎與核心命題**：定義動機的先驗性、唯一性與宇宙初始條件（目的參數）。
2. **MQS 評估系統家族**：系統化梳理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具的職能與邊界。
3. **自省系統與代理實踐**：探討 Reflexive System、雙核對齊與 M-SOS 的實作機制。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與動機論的本體論地位。
5. **動機論知識總覽**：總結上述架構，形成代理行為的決策模型。
