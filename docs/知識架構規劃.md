# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯脈絡的知識體系，作為動機論的實踐指南。

## 知識架構藍圖
1. **動機論基礎與核心命題**：定義動機的先驗性、唯一性與宇宙初始條件（目的參數）。
2. **MQS 評估系統家族**：系統化整理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估指標，建立動機核的品質檢驗標準。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊，說明代理如何維持動機一致性。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與動機論的本體論地位。
