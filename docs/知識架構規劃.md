# 知識架構規劃：動機論體系化

本文件旨在將 32 篇閱讀筆記重整為具備邏輯層次的知識體系，作為動機核治理的理論基石。

## 預計架構
1. **動機論基礎與核心命題**：定義動機、動機核、先驗性與唯一性原則。
2. **MQS 評估系統家族**：系統化整理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具的職能。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊的實作機制。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與目的參數，探討動機的本體論地位。
