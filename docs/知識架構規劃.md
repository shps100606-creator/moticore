# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為系統性的知識體系，作為動機論的實踐指南。

## 架構規劃
1. **01-動機論基礎與核心命題**：定義動機、動機核、Motivum 與動機先驗性。
2. **02-MQS評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具的邏輯。
3. **03-自省系統與代理實踐**：探討 Reflexive System、雙核對齊與 M-SOS 的實際運作。
4. **04-動機論哲學基礎**：連結 Psi 理論、CogAff 架構與目的參數，確立本體論地位。
5. **動機論知識總覽**：總結動機治理 AI 的可能性與限制。

## 執行策略
