# 知識架構規劃：動機論體系重組

本文件旨在將 32 篇閱讀筆記轉化為結構化的知識體系，以驗證動機核的有效性。

## 知識架構藍圖
1. **動機論基礎與核心命題**：定義動機、動機核、Motivum 與動機唯一性原則。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具，建立品質控制標準。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊機制。
4. **動機論哲學基礎**：探討動機先驗性、目的參數與 Psi 理論的跨學科連結。
