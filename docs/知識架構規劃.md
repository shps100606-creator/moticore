本文件旨在將 32 篇閱讀筆記重整為具備邏輯層次的知識體系，作為動機核治理的理論基石。

## 知識架構藍圖
1. **第一層：動機論基礎與核心命題**
   - 核心概念：動機先驗性、目的參數、動機唯一性原則。
   - 目的：確立動機作為宇宙運算初始條件的本體論地位。

2. **第二層：MQS 評估系統家族**
   - 核心概念：MNQS, MPQS, MAQS, MDQS, MEQS, MIQS。
   - 目的：建立動機核的品質檢測與自我審查標準。

3. **第三層：自省系統與代理實踐**
   - 核心概念：Reflexive System, 語義裁決庫 (SDI), 雙核對齊, M-SOS。
   - 目的：將抽象動機轉化為代理的日常運作邏輯。

4. **第四層：動機論哲學基礎**
   - 核心概念：Psi 理論、CogAff 架構、必要公理、負向先驗。
   - 目的：探討動機論與認知科學、計算理論的交叉點。
