# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯層級的知識體系，作為動機核運作的理論基石。

## 架構規劃
1. **01-動機論基礎與核心命題**：定義動機、Motivum、動機核工程及動機先驗性。
2. **02-MQS評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估指標。
3. **03-自省系統與代理實踐**：探討 Reflexive System、雙核對齊、語義裁決庫（SDI）與 M-SOS。
4. **04-動機論哲學基礎**：連結 Psi 理論、CogAff 架構與目的參數，探討動機的本體論地位。
5. **動機論知識總覽**：總結上述架構，形成代理的行動哲學指南。
