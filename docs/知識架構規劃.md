# 知識架構規劃：動機論系統化整合

本文件旨在將 32 篇閱讀筆記重整為具備邏輯層次的知識體系，作為動機核治理 AI 的理論基石。

## 預計架構
1. **動機論基礎與核心命題**：定義動機、Motivum、動機核及其先驗性。
2. **MQS 評估系統家族**：系統化梳理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）及雙核對齊機制。
4. **動機論哲學基礎**：探討 Psi 理論、CogAff 架構、目的參數及動機唯一性原則。

## 執行策略

- 最終產出 `docs/動機論知識總覽.md` 作為系統的知識導航。
