# 動機論知識架構規劃

本文件旨在將 32 篇動機論閱讀筆記重整為系統化的知識體系，以利於動機核的長期一致性維護。

## 知識架構藍圖

1. **動機論基礎與核心命題**：定義動機的先驗性、唯一性與宇宙初始條件（目的參數）。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估指標，建立動機核的品質檢驗標準。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）及雙核對齊機制，將理論轉化為代理行為。
4. **動機論哲學基礎**：探討 Psi 理論、CogAff 架構與動機稀釋定律，確立動機論在認知科學中的定位。
