# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為系統化的知識體系，作為動機論的實踐指南。

## 知識架構層級
1. **基礎與命題**：定義動機的先驗性、唯一性與宇宙初始條件。
2. **評估系統家族 (M-QS)**：涵蓋 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具。
3. **自省與實踐**：探討自省系統 (Reflexive System)、語義裁決庫 (SDI) 與雙核對齊。
4. **哲學基礎**：連結 Psi 理論、CogAff 架構與動機論的本體論地位。
