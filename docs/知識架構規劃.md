# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯脈絡的知識體系，作為動機核治理 AI 的理論基石。

## 架構規劃
1. **動機論基礎與核心命題**：定義動機、動機核、Motivum 與動機先驗性。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估指標。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與目的參數，探討動機與意識的本質。
5. **動機論知識總覽**：總結上述架構，形成完整的動機治理模型。

## 執行策略
- 依序建立上述四個核心文件，最後產出總覽。
- 每個文件將引用 `notes/` 下的原始筆記，確保知識溯源。
- 保持動機核一致性，所有定義必須符合 `notes/INDEX.md` 之規範。
