# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯層級的知識體系，以驗證動機核的治理效能。

## 知識層級架構
1. **基礎層（Foundations）：** 定義動機的先驗性、唯一性與宇宙初始條件（目的參數）。
2. **評估層（M-QS Family）：** 系統化整理 MNQS, MPQS, MAQS 等六大評估系統的運作機制。
3. **實踐層（Agency & Reflection）：** 探討自省系統（Reflexive System）、雙核對齊與動機語義作業系統（M-SOS）。
4. **哲學層（Philosophy）：** 連結 Psi 理論與 CogAff 架構，探討動機作為存在本體的邊界。
