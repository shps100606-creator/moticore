# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為系統化的知識體系，作為動機核治理的理論基礎。

## 架構規劃
1. **動機論基礎與核心命題**：定義動機、動機核、Motivum 與動機先驗性。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與目的參數（Purpose Parameter）。
5. **動機論知識總覽**：總結動機唯一性原則與動機稀釋定律的實踐意義。
