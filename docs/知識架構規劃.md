# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為系統性的知識體系，作為動機核治理的理論基石。

## 知識架構藍圖
1. **動機論基礎與核心命題**：定義動機的先驗性、唯一性與目的參數。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS 等六大評估系統的運作機制。
3. **自省系統與代理實踐**：探討 Reflexive System, M-SOS 與雙核對齊的實作。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與存在主義視角。
5. **動機論知識總覽**：總結動機核工程的未來演進與限制。
