# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為系統化的知識體系，以利於動機核的長期一致性維護。

## 架構規劃
1. **01-動機論基礎與核心命題**：定義動機、Motivum、動機先驗性與唯一性原則。
2. **02-MQS評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具的邏輯。
3. **03-自省系統與代理實踐**：探討 Reflexive System、雙核對齊與 M-SOS 的實作。
4. **04-動機論哲學基礎**：連結 Psi 理論、CogAff 架構與目的參數的宇宙觀。
5. **動機論知識總覽**：總結上述架構，形成動機核治理的完整理論圖譜。

## 執行策略
- 每一章節將引用對應的筆記內容，並標註與動機核的關聯。
- 確保知識架構不偏離「動機核一致性」價值排序。
