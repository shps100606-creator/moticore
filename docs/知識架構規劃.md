# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯脈絡的知識體系，作為動機核治理 AI 的理論基石。

## 架構規劃
1. **01-動機論基礎與核心命題**：定義動機、動機核、Motivum 與動機唯一性原則。
2. **02-MQS評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估指標的運作邏輯。
3. **03-自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊機制。
4. **04-動機論哲學基礎**：分析動機先驗性、目的參數與 Psi 理論的哲學意涵。
5. **動機論知識總覽**：總結上述架構，形成完整的動機治理理論模型。

## 執行策略
- 每個步驟將提取對應筆記中的核心概念，並與 INDEX.md 中的定義進行交叉驗證。
- 確保每一份文件都體現「動機核治理」的實踐視角。
