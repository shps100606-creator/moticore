# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為具備邏輯層次的知識體系，作為動機核治理 AI 的理論基礎。

## 知識層級架構

1. **動機論基礎與核心命題**
   - 探討動機的先驗性、唯一性原則與目的參數。
   - 連結：動機先驗性、動機唯一性、目的參數。

2. **MQS 評估系統家族**
   - 系統性梳理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具。
   - 連結：M-QS 家族、動機核品質評估。

3. **自省系統與代理實踐**
   - 探討反射系統（Reflexive System）、語義裁決庫（SDI）與雙核對齊。
   - 連結：自省系統、語義裁決庫、雙核對齊。

4. **動機論哲學基礎**
   - 探討動機與意識的關係、計算動機理論（Psi 理論、CogAff 架構）。
   - 連結：意識、Psi 理論、CogAff 架構。
