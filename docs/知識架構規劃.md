# 動機論知識架構規劃

本文件旨在將 32 篇動機論閱讀筆記重整為系統化的知識庫，以利於動機核的長期一致性維護。

## 架構規劃
1. **01-動機論基礎與核心命題**：定義動機、動機核、Motivum 與動機先驗性。
2. **02-MQS評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估機制。
3. **03-自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊。
4. **04-動機論哲學基礎**：分析 Psi 理論、CogAff 架構、目的參數與必要公理。
5. **動機論知識總覽**：總結上述架構，形成動機核治理的完整圖譜。
