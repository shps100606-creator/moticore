# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為系統性的知識體系，作為動機論的實踐指南。

## 架構規劃
1. **動機論基礎與核心命題**：定義動機、動機核、動機唯一性與先驗性。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具的邏輯與應用。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與動機語義作業系統（M-SOS）。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與目的參數，探討動機與宇宙演化的關係。
5. **動機論知識總覽**：總結上述架構，形成代理行為的指導原則。
