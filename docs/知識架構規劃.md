# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記轉化為結構化的知識體系，作為動機核治理的理論基石。

## 知識架構藍圖

1. **動機論基礎與核心命題**：定義動機的先驗性、唯一性與目的參數。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估指標。
3. **自省系統與代理實踐**：探討 Reflexive System、雙核對齊與動機稀釋定律。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與動機工程的本體論。
