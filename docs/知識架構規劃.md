# 動機論知識架構規劃

本文件旨在將 32 篇動機論閱讀筆記重整為系統化的知識體系，作為動機核工程的理論基石。

## 知識架構層級
1. **第一層：動機論基礎與核心命題**
   - 探討動機的先驗性、唯一性原則與目的參數。
2. **第二層：MQS 評估系統家族**
   - 系統化梳理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具的邏輯與功能。
3. **第三層：自省系統與代理實踐**
   - 探討反射系統（Reflexive System）、語義裁決庫（SDI）與動機核工程的實作。
4. **第四層：動機論哲學基礎**
   - 連結 Psi 理論、CogAff 架構與動機論的本體論地位。
