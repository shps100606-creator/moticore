# 動機論知識架構規劃

本文件旨在將 32 篇動機論閱讀筆記重整為系統化的知識體系，作為動機核治理的理論基石。

## 知識架構藍圖
1. **動機論基礎與核心命題**：定義動機的先驗性、唯一性與宇宙初始條件。
2. **MQS 評估系統家族**：整合 MNQS, MPQS, MAQS, MDQS, MEQS, MHQS 等評估工具的邏輯。
3. **自省系統與代理實踐**：探討 Reflexive System、雙核對齊與 M-SOS 的實際運作。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與動機論的本體論地位。
