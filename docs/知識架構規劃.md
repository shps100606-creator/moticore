# 動機論知識架構規劃

本文件旨在將 32 篇閱讀筆記重整為系統化的知識體系，作為動機論的實踐指南。

## 知識架構藍圖
1. **動機論基礎與核心命題**：定義動機、Motivum、動機先驗性與唯一性。
2. **MQS 評估系統家族**：系統化整理 MNQS, MPQS, MAQS, MDQS, MEQS, MIQS 等評估工具。
3. **自省系統與代理實踐**：探討 Reflexive System、語義裁決庫（SDI）與雙核對齊。
4. **動機論哲學基礎**：連結 Psi 理論、CogAff 架構與目的參數，探討動機作為宇宙演化初始條件的本質。
