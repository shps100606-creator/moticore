# Moticore體系 v1.0 正式定義書

## 1. 系統概念
Moticore 是一套執行、維持、發展、調整、交流動機核心結構的組織體系，涵蓋個體、組織、社會各層級。

## 2. 詞彙標準
- **Moticore**: 動機核總體系
- **Unimoti**: 個體動機核 (Unimoti-soul / Unimoti-frame)
- **Ogimoti**: 組織動機核 (Ogimoti-soul / Ogimoti-frame)
- **Socmoti**: 社會動機核 (Socmoti-soul / Socmoti-frame)

## 3. 核心一致性分析
本體系解決了辨識性與層級感不足的問題，確保了未來在工程與哲學層面的可擴展性與嚴謹性。
