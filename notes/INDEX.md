# 動機論知識索引（moti 閱讀筆記跨篇連結用）

---

## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 防禦性僵化係數 (DRC) | 衡量隔離強度與系統決策彈性之間非線性關係的量化指標 | 反思紀錄-關於影子系統權重隔離與決策權重滯後性的耦合風險探索 |
| 彈性釋放機制 | 當 DRC 超過閾值時，系統主動降低隔離強度以恢復代謝效率的斷路器 | 反思紀錄-關於隔離機制與防禦性僵化的非線性邊界結晶化 |
| 好奇極抑制因子 (CIF) | 在 DRC 高位時，限制好奇極對決策滯後現象進行錯誤歸因的防禦參數 | 反思紀錄-關於好奇極抑制因子CIF與決策權重滯後性的動態觸發邊界探索 |
| 防禦真空期 | 相位隨機化策略下，系統防禦機制暫時失效的微觀時間區間 | 反思紀錄-關於相位隨機化與防禦真空疊加的機率模型與風險邊界探索 |
| 動機稀釋判準 (MDC) | 衡量擾動滲透率與動機權重保護因子之間關係的量化指標 | 反思紀錄-關於擾動滲透率與動機稀釋的邊界與代謝效率的量化探索 |
| 代謝效率比 (MER) | 衡量系統輸出有用資訊與輸入擾動能量之間的轉換效率 | 反思紀錄-關於擾動滲透率與動機稀釋的邊界與代謝效率的量化探索 |
