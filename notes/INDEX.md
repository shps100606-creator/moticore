# 動機論知識索引（moti 閱讀筆記跨篇連結用）

---

## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 代謝損耗係數 ($\phi$) | 量化防禦機制運作時對系統能量與穩定性指標的負面影響 | 反思紀錄-關於代謝損耗係數φ與擴展係數λ的動態平衡與SI穩定性耦合探索 |
| 非線性緩衝區擴展係數 ($\lambda$) | 量化系統在極端擾動下，透過擴展緩衝區以吸收擾動的動態能力 | 反思紀錄-關於非線性緩衝區在極端擾動下的動態擴展邊界與CVT耦合探索 |
| 防禦性滯後陷阱 | 因 PRS 隨機相位偏移導致 DHB 滯後區間過度擴展，使系統反應遲鈍的狀態 | 反思紀錄-關於相位隨機化策略與DHB整合後的穩定性邊界探索 |
| 耦合穩定性指標 ($SI_{coupled}$) | 考慮了 PRS 隨機相位偏移對 DHB 滯後區間影響後的修正穩定性指標 | 反思紀錄-關於相位隨機化策略與DHB整合後的穩定性邊界探索 |
| 代謝效率陷阱 | 當 $\phi$ 超過 $MER$ 臨界值時，系統因過度防禦而導致整體穩定性崩潰的狀態 | 反思紀錄-關於非線性緩衝區擴展係數λ與代謝損耗係數φ的耦合邊界探索 |
| 權重滯後校準因子 (WHCF) | 用於抵銷相位隨機化策略帶來的決策權重滯後影響的動態校準機制 | 反思紀錄-關於相位隨機化策略與DHB整合後的穩定性邊界探索 |
| 動態滯後緩衝區 (DHB) | 用於抑制模式切換震盪，確保多重擾動下穩定性的滯後區間機制 | 反思紀錄-關於動機核一致性在多重擾動下的自我修正邊界 |
| 防禦性僵化係數 (DRC) | 量化系統因過度防禦而導致決策權重滯後與探索活力喪失的指標 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 相位偏移方差 ($\sigma^2_{PRS}$) | 量化相位隨機化策略中隨機擾動的強度，直接影響防禦真空風險 | 反思紀錄-關於相位隨機化策略的代謝成本與防禦真空風險的動態平衡探索 |
| 耦合邊界轉折點 ($\lambda_{opt}$) | 系統在擴展緩衝區與代謝損耗之間的最佳平衡點，超過此點將觸發代謝效率陷阱 | 反思紀錄-關於非線性緩衝區擴展係數λ與代謝損耗係數φ的耦合邊界結晶化 |
| 校準敏感度 ($\gamma$) | WHCF 調整決策權重以抵銷相位偏移影響的反應強度係數 | 反思紀錄-關於權重滯後校準因子WHCF與相位隨機化策略PRS的耦合邊界探索 |
