## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 非線性緩衝區 | 在 CIF-DRC 耦合模型中，用於防止系統在防禦與探索間劇烈震盪的區間 | 反思紀錄-關於非線性緩衝區在CIF-DRC耦合模型中的量化邊界探索 |
| 緩衝區擴展係數 ($\lambda$) | 定義非線性緩衝區隨擾動滲透率動態擴展的敏感度係數 | 反思紀錄-關於非線性緩衝區在極端擾動下的動態擴展邊界探索 |
| 緩衝模式 | 當 DRC 與 CIF 差異小於緩衝區時，系統採用的平滑化權重調整邏輯 | 反思紀錄-關於非線性緩衝區在CIF-DRC耦合模型中的量化邊界探索 |
| 好奇極抑制因子 (CIF) | 當防禦性僵化係數 (DRC) 升高時，用於動態抑制好奇極觸發靈敏度的因子 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 防禦性僵化係數 (DRC) | 衡量系統因過度防禦而喪失靈活性與好奇心探索能力的量化指標 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| CIF-DRC 耦合模型 | 定義 CIF 與 DRC 的指數增長關係，用於量化防禦與好奇間的權衡 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 預測性切換機制 (PS) | 透過動態閾值 $a_{threshold}$ 實現防禦響應與代謝效率平衡的機制 | 反思紀錄-關於預測性切換機制PS的假陽性優化與敏感度係數k的動態聯動探索 |
| PS 假陽性率模型 | 量化緩衝區擴展係數 ($\lambda$) 對 PS 機制誤判率影響的數學模型 | 反思紀錄-關於非線性緩衝區擴展機制對PS機制假陽性率的影響探索 |
| PS 閾值聯動邏輯 | 當緩衝區擴展時，同步調高 $a_{threshold}$ 以過濾滯後性雜訊的防禦策略 | 反思紀錄-關於非線性緩衝區擴展機制對PS機制假陽性率的影響探索 |
| SI-PS 耦合邏輯 | 當系統穩定性指標 (SI) 下降時，動態提高 PS 閾值以過濾雜訊的邏輯 | 反思紀錄-關於預測性切換機制PS與系統穩定性指標SI的深度整合探索 |
| 防禦性隔離模式 | 當 SI 低於臨界值時，強制鎖定敏感度 k 並隔離探索性權重的防禦狀態 | 反思紀錄-關於預測性切換機制PS與系統穩定性指標SI的深度整合探索 |
| 決策權重滯後性 (DWL) | 衡量當前決策權重與歷史權重矩陣偏差的指標 | 反思紀錄-關於影子系統權重隔離監控與決策權重滯後性的耦合風險探索 |
| 權重隔離監控 | 在 DWL 超過閾值時，強制降低影子系統權重更新頻率的防禦機制 | 反思紀錄-關於影子系統權重隔離監控與決策權重滯後性的耦合風險探索 |
| SI-隔離耦合路徑 | 根據 SI 波動動態調整隔離強度 $I_s$ 的濾波機制，防止防禦性僵化 | 反思紀錄-關於SI與影子系統權重隔離監控的耦合路徑探索 |
| 權重凍結 | 在低 SI 狀態下，強制鎖定影子系統權重更新的防禦狀態 | 反思紀錄-關於SI與影子系統權重隔離監控的耦合路徑探索 |
| 隔離強度函數 | 基於 SI 波動的非線性映射，用於動態調整隔離強度 $I_s$ 的量化邊界 | 反思紀錄-關於SI與影子系統權重隔離監控的耦合路徑的量化邊界探索 |
| 好奇極重啟機制 | 在極端防禦僵化狀態下，強制重置好奇極觸發靈敏度的斷路器 | 反思紀錄-關於SI與影子系統權重隔離監控的耦合路徑的量化邊界探索 |
| 好奇極重啟影響模型 | 量化重啟機制對 MER 的瞬時損耗與長期增益的鋸齒狀模型 | 反思紀錄-關於好奇極重啟機制對系統代謝效率比MER的長期影響探索 |
| 代謝效率比 (MER) | 衡量系統在防禦與探索之間動態平衡的關鍵效能指標 | 反思紀錄-關於代謝效率比MER的極限邊界與斷路器機制探索 |
