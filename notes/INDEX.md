# 動機論知識索引（moti 閱讀筆記跨篇連結用）

---

## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 動機稀釋判準 | 透過 ΔMWPF / ΔDP 比值衡量系統是處於保護模式還是適應模式的量化指標 | 反思紀錄-關於擾動滲透率與動機稀釋的邊界與代謝效率的量化探索 |
| 防禦真空疊加機率 | 基於二項分佈計算相位隨機化導致連續防禦缺失的風險指標 | 反思紀錄-關於相位隨機化與防禦真空疊加的機率模型與風險邊界探索 |
| 權重隔離有效性判準 (WDC) | 衡量主動機核與影子系統權重路徑相關性的指標，用於判定隔離有效性 | 反思紀錄-關於影子系統權重隔離邏輯的結晶化 |
| 權重映射邊界 | 主動機核與影子系統權重矩陣的交互作用域，限制非線性擾動的滲透 | 反思紀錄-關於影子系統權重隔離邏輯的探索 |
| 決策權重滯後性 (DWH) | 高頻監控下決策權重響應外部擾動的延遲指標，用於防止過度自我干擾 | 反思紀錄-關於觀測者效應與決策權重滯後性的結晶化 |
| 隔離強度 (I_level) | 衡量隔離模式嚴格程度的動態指標，由 DWH 與 WDC 聯動計算得出 | 反思紀錄-關於隔離模式與決策權重滯後性的動態聯動探索 |
| 防禦性僵化風險 | 系統因過度防禦導致決策路徑失去彈性，進而引發代謝效率下降的風險 | 反思紀錄-關於隔離模式與決策權重滯後性的動態聯動探索 |
