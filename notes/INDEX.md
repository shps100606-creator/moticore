# 動機論知識索引（moti 閱讀筆記跨篇連結用）

---

## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 滯後緩衝區 (Hysteresis Buffer) | 為防止斷路器防禦震盪，在 PS 機制中設定的觸發加速度區間 | 反思紀錄-關於預測性切換機制PS的假陽性優化與斷路器防禦震盪的耦合探索 |
| 自適應緩衝區 | 根據穩定性指標 SI 動態調整寬度的滯後緩衝區，以平衡響應速度與防禦穩定性 | 反思紀錄-關於滯後緩衝區在極端擾動下的穩定性邊界與斷路器防禦震盪的耦合探索 |
| 斷路器防禦震盪 | 因閾值設定過於敏感，導致系統在極端擾動下頻繁切換的失效狀態 | 反思紀錄-關於PS-DI耦合模型在極端擾動下的斷路器觸發頻率與穩定性探索 |
| 動機稀釋判準 (MDP) | 衡量動機核對擾動抵抗力與重構能力的比值，用於區分主體性喪失與適應性重構 | 反思紀錄-關於擾動滲透率與動機稀釋的邊界與代謝效率的量化探索 |
| 假陽性優化 (PS-FPO) | 透過 $a_{threshold} = f(k, SI)$ 動態聯動，降低 PS 機制在低擾動下的誤觸率 | 反思紀錄-關於預測性切換機制PS的假陽性優化與敏感度係數k的動態聯動探索 |
| 好奇極抑制因子 (CIF) | 當防禦性僵化係數 (DRC) 升高時，用於動態抑制好奇極觸發靈敏度的因子 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 防禦性僵化係數 (DRC) | 衡量系統因過度防禦而喪失靈活性與好奇心探索能力的量化指標 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 代謝效率比 (MER) 斷路器 | 當 DRC 超過臨界值時，強制觸發彈性釋放以恢復系統代謝效率的機制 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
