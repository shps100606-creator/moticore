# 動機論知識索引（moti 閱讀筆記跨篇連結用）

---

## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 防禦性僵化係數 (DRC) | 衡量隔離強度與系統決策彈性之間非線性關係的量化指標 | 反思紀錄-關於影子系統權重隔離與決策權重滯後性的耦合風險探索 |
| 彈性釋放機制 | 當 DRC 超過閾值時，系統主動降低隔離強度以恢復代謝效率的斷路器 | 反思紀錄-關於隔離機制與防禦性僵化的非線性邊界結晶化 |
| 好奇極抑制因子 (CIF) | 在 DRC 高位時，限制好奇極對決策滯後現象進行錯誤歸因的防禦參數 | 反思紀錄-關於好奇極抑制因子CIF與決策權重滯後性的動態觸發邊界探索 |
| 認知真空風險 (CVR) | 因過度抑制好奇極而導致系統喪失對外部非線性擾動捕捉能力的風險狀態 | 反思紀錄-關於認知真空風險與好奇極抑制因子CIF的動態耦合探索 |
| 穩定性指標 (SI) | 衡量系統在防禦與探索之間動態平衡的綜合量化基準 | 反思紀錄-關於認知真空風險與好奇極抑制因子CIF的動態耦合探索 |
| 好奇極重啟機制 | 當 CVR 超過閾值時，強制繞過 CIF 抑制以恢復環境觀測的保護機制 | 反思紀錄-關於認知真空風險與好奇極抑制因子CIF的動態耦合探索 |
| 防禦慣性 (DI) | 系統在模式切換時因權重重校準而產生的滯後效應 | 反思紀錄-關於防禦慣性與預測性切換機制的耦合效應探索 |
| 預測性切換機制 (PS) | 透過監控擾動加速度提前進行權重預分配的輔助斷路器 | 反思紀錄-關於防禦慣性與預測性切換機制的耦合效應探索 |
| 相位偏移約束 | 透過隨機相位偏移強制拉開防禦切換節點，以降低防禦真空疊加機率的策略 | 反思紀錄-關於相位隨機化與防禦真空疊加的機率模型與風險邊界探索 |
| 防禦真空疊加風險 | 在多重防禦機制並行時，因相位同步導致防禦空窗期連續發生的機率風險 | 反思紀錄-關於相位隨機化與防禦真空疊加的機率模型與風險邊界探索 |
