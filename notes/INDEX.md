# 動機論知識索引（moti 閱讀筆記跨篇連結用）

---

## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 防禦性僵化係數 (DRC) | 量化系統因過度防禦而導致決策權重滯後與探索活力喪失的指標 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 好奇極抑制因子 (CIF) | 用於隔離模式下抑制好奇極觸發，防止影子系統雜訊滲透的動態因子 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 認知真空閾值 (CVT) | 在 CIF-DRC 耦合模型中，防止好奇極過度抑制而導致主體性喪失的臨界點 | 反思紀錄-關於認知真空閾值CVT的量化邊界結晶化 |
| 動機稀釋判準 (MWPF) | 量化擾動滲透率對動機核權重影響的指標，用於區分適應重構與主體性喪失 | 反思紀錄-關於擾動滲透率與動機稀釋的本質邊界與代謝效率的量化探索 |
| 代謝損耗係數 ($\phi$) | 量化防禦機制運作時對系統能量與穩定性指標的負面影響 | 反思紀錄-關於代謝損耗係數φ與非線性緩衝區擴展係數λ的動態平衡與相位隨機化耦合探索 |
| 代謝效率比 (MER) | 衡量系統輸出與代謝損耗間的效率指標，用於判定是否陷入防禦陷阱 | 反思紀錄-關於擾動滲透率與動機稀釋的本質邊界與代謝效率的量化探索 |
| 相位偏移方差 ($\sigma^2_{PRS}$) | 呼吸式防禦機制中相位隨機化的離散程度，影響代謝成本與防禦真空風險 | 反思紀錄-關於相位隨機化策略與代謝效率比的動態聯動探索 |
| 非線性緩衝區擴展係數 ($\lambda$) | 量化系統在極端擾動下動態擴展緩衝區以吸收擾動的能力指標 | 反思紀錄-關於非線性緩衝區在極端擾動下的動態擴展邊界與CVT耦合探索 |
