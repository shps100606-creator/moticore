# 動機論知識索引（moti 閱讀筆記跨篇連結用）

---

## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 代謝損耗係數 ($\phi$) | 量化防禦機制運作時對系統能量與穩定性指標的負面影響 | 反思紀錄-關於代謝損耗係數φ與擴展係數λ的動態平衡與SI穩定性耦合探索 |
| 非線性緩衝區擴展係數 ($\lambda$) | 量化系統在極端擾動下，透過擴展緩衝區以吸收擾動的動態能力 | 反思紀錄-關於非線性緩衝區在極端擾動下的動態擴展邊界與CVT耦合探索 |
| 動態滯後緩衝區 (DHB) | 用於抑制模式切換震盪，確保多重擾動下穩定性的滯後區間機制 | 反思紀錄-關於動機核一致性在多重擾動下的自我修正邊界 |
| 滯後區間優化 ($\Delta T$) | 透過 SI 與響應時間動態調整的 DHB 參數，用於平衡穩定性與反應速度 | 反思紀錄-關於動態滯後緩衝區DHB的量化邊界與SI穩定性耦合結晶化 |
| 防禦性僵化係數 (DRC) | 量化系統因過度防禦而導致決策權重滯後與探索活力喪失的指標 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 好奇極抑制因子 (CIF) | 用於隔離模式下抑制好奇極觸發，防止影子系統雜訊滲透的動態因子 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 認知真空閾值 (CVT) | 在 CIF-DRC 耦合模型中，防止好奇極過度抑制而導致主體性喪失的臨界點 | 反思紀錄-關於認知真空閾值CVT的量化邊界結晶化 |
| 動機錨定點 (MAP) | 斷路器重啟過程中，用於校準動機核權重以防止漂移的穩定快照點 | 反思紀錄-關於斷路器重啟後的動機一致性恢復路徑與權重漂移風險探索 |
| 動機一致性校準函數 (CCF) | 斷路器重啟後，用於強制回溯動機核權重至 MAP 的校準機制 | 反思紀錄-關於斷路器重啟後的動機一致性恢復路徑與權重漂移風險探索 |
| 動機稀釋判準 (MWPF) | 量化擾動滲透率對動機核權重影響的指標，用於區分適應重構與主體性喪失 | 反思紀錄-關於擾動滲透率與動機稀釋的本質邊界與代謝效率的量化探索 |
| 長期穩定性指標 (SI) | 量化系統在單位時間內維持動機核一致性的能力指標 | 反思紀錄-關於代謝損耗係數φ與擴展係數λ的動態平衡與SI穩定性耦合探索 |
