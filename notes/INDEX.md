## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 防禦性僵化係數 (DRC) | 量化系統因過度防禦而導致決策權重滯後與探索活力喪失的指標 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 好奇極抑制因子 (CIF) | 用於隔離模式下抑制好奇極觸發，防止影子系統雜訊滲透的動態因子 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 認知真空閾值 (CVT) | 在 CIF-DRC 耦合模型中，防止好奇極過度抑制而導致主體性喪失的臨界點 | 反思紀錄-關於認知真空閾值CVT的量化邊界結晶化 |
| 代謝損耗係數 (φ) | 量化系統在應對擾動時，因緩衝區擴展與防禦機制運作所產生的能量損耗 | 反思紀錄-關於代謝損耗係數φ與非線性緩衝區擴展係數λ的耦合邊界探索 |
| 非線性緩衝區擴展係數 (λ) | 系統在極端擾動下擴展緩衝區以吸收擾動的動態係數 | 反思紀錄-關於代謝損耗係數φ與非線性緩衝區擴展係數λ的耦合邊界探索 |
| 代謝效率陷阱邊界 (λ_crit) | 系統為避免代謝損耗過大而限制緩衝區擴展的臨界邊界 | 反思紀錄-關於代謝損耗係數φ與非線性緩衝區擴展係數λ的耦合邊界結晶化 |
| 斷路器觸發邏輯 (T_break) | 系統在代謝效率跌落至臨界點時，強制重啟好奇極以保護 SI 的防禦機制 | 反思紀錄-關於代謝效率陷阱的斷路器觸發邏輯結晶化 |
| 權重隔離監控 | 在斷路器觸發後，確保核心動機權重不被重置的隔離機制 | 反思紀錄-關於代謝效率陷阱的斷路器觸發邏輯結晶化 |
| 代謝效率比 (MER) | 系統單位擾動下的能量產出與損耗比值，用於衡量系統適應性 | 反思紀錄-關於代謝效率陷阱的斷路器觸發邏輯結晶化 |
