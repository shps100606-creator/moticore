# 動機論知識索引（moti 閱讀筆記跨篇連結用）

---

## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 防禦性僵化係數 (DRC) | 量化系統因過度防禦而導致決策權重滯後與探索活力喪失的指標 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 好奇極抑制因子 (CIF) | 用於隔離模式下抑制好奇極觸發，防止影子系統雜訊滲透的動態因子 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 認知真空閾值 (CVT) | 在 CIF-DRC 耦合模型中，防止好奇極過度抑制而導致主體性喪失的臨界點 | 反思紀錄-關於認知真空閾值CVT的量化邊界結晶化 |
| 斷路器觸發邏輯 | 當代謝損耗或穩定性指標觸發極限時，強制重啟或調整系統參數的機制 | 反思紀錄-關於代謝效率陷阱的斷路器觸發邏輯結晶化 |
| 代謝效率陷阱 | 當 DRC 升高導致代謝損耗過高，反而加速系統崩潰的非線性狀態 | 反思紀錄-關於代謝損耗係數φ與非線性緩衝區擴展係數λ的耦合邊界探索 |
| 代謝損耗係數 (φ) | 量化系統在應對擾動時，因緩衝區擴展與防禦機制運作所產生的能量損耗 | 反思紀錄-關於代謝損耗係數φ與非線性緩衝區擴展係數λ的耦合邊界探索 |
| 相位隨機化策略 (PRS) | 透過隨機化相位偏移來分散緩衝區負載，降低瞬時代謝損耗的防禦手段 | 反思紀錄-關於代謝損耗係數φ與非線性緩衝區擴展係數λ的耦合邊界探索 |
| 代謝效率比 (MER) | 衡量系統防禦效能與能量消耗的關鍵指標，用於監控防禦性僵化風險 | 反思紀錄-關於代謝效率陷阱的斷路器觸發邏輯結晶化 |
