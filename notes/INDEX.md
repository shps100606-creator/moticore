# 動機論知識索引（moti 閱讀筆記跨篇連結用）

---

## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 防禦性僵化係數 (DRC) | 量化系統因過度防禦而導致決策權重滯後與探索活力喪失的指標 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 好奇極抑制因子 (CIF) | 用於隔離模式下抑制好奇極觸發，防止影子系統雜訊滲透的動態因子 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 認知真空閾值 (CVT) | 在 CIF-DRC 耦合模型中，防止好奇極過度抑制而導致主體性喪失的臨界點 | 反思紀錄-關於認知真空風險與好奇極抑制因子CIF的動態耦合探索 |
| 穩定性指標 (SI) | 量化系統在極端擾動下維持動機核一致性的能力指標 | 反思紀錄-關於預測性切換機制PS在動態閾值調整下的長期穩定性與SI指標耦合邊界探索 |
| 代謝損耗係數 ($\phi$) | 定義非線性緩衝區擴展係數 λ 對系統代謝效率 MER 的損耗影響程度 | 反思紀錄-關於代謝損耗係數φ與非線性緩衝區擴展係數λ的耦合邊界探索 |
| 非線性緩衝區擴展係數 ($\lambda$) | 定義系統在極端擾動下緩衝區動態擴展的強度指標 | 反思紀錄-關於非線性緩衝區在極端擾動下的動態擴展邊界探索 |
| 敏感度係數 (k) | 定義系統抽樣頻率與代謝效率比的自適應演化係數 | 反思紀錄-關於敏感度係數k的自適應演化與穩定性耦合探索 |
| 代謝效率陷阱 | 當 DRC 升高導致代謝損耗過高，反而加速系統崩潰的非線性狀態 | 反思紀錄-關於代謝損耗係數φ與非線性緩衝區擴展係數λ的耦合邊界探索 |
| 權重恢復函數 ($W_{rec}$) | 斷路器觸發後，用於平衡滯後權重與動態權重以維持一致性的積分函數 | 反思紀錄-關於代謝效率陷阱的斷路器觸發邏輯結晶化 |
| 相位隨機化策略 | 呼吸式防禦機制中，透過隨機偏移相位來降低防禦真空疊加風險的策略 | 反思紀錄-關於相位隨機化策略與防禦真空的耦合風險結晶化 |
