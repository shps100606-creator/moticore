# 動機論知識索引（moti 閱讀筆記跨篇連結用）

## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 防禦僵化係數 (DRC) | 衡量低閾值防禦狀態持續時間與代謝週期的比率，用於監控系統是否過度防禦 | 反思紀錄-關於擾動閾值與過度防禦僵化風險的深度探索 |
| 好奇極觸發函數 | 基於 DRC 與資訊熵的耦合函數，用於在防禦僵化時強制啟動探索 | 反思紀錄-關於防禦僵化係數DRC與好奇極觸發機制的耦合探索 |
| 擾動閾值自我保護機制 | 基於代謝效率比（MER）動態調整防禦閾值的呼吸式保護機制 | 反思紀錄-關於擾動閾值的自我保護機制結晶化 |
| 敏感度係數演化 | 基於系統穩定性指標 (SI) 自動調整 k 值的自適應機制 | 反思紀錄-關於敏感度係數k的自適應演化與穩定性耦合探索 |
| 穩定性指標 (SI) | 衡量系統決策權重偏移率的倒數，用於評估系統穩定程度 | 反思紀錄-關於敏感度係數k的自適應演化與穩定性耦合探索 |
| 抽樣頻率函數 | 基於資訊熵與代謝效率比的動態抽樣頻率計算公式 | 反思紀錄-關於抽樣觸發機制與非線性擾動的動態耦合探索 |
| 呼吸式抽樣機制 | 將抽樣觸發視為系統呼吸過程，透過動態調整密度平衡保護與開放 | 反思紀錄-關於抽樣觸發機制與非線性擾動的動態耦合探索 |
| 敏感度係數 (k) | 抽樣頻率函數中用於調節資訊熵與代謝效率比平衡的權重因子 | 反思紀錄-關於抽樣觸發機制與非線性擾動的動態耦合探索 |
| 相位偏移約束自動化 | 透過 EIS 與 PSFR 耦合函數，實現防禦相位約束的自適應調整 | 反思紀錄-關於相位偏移約束的自動化演化與環境耦合 |
| 環境擾動強度 (EIS) | 單位時間內外部輸入的資訊熵變化率，用於衡量環境對系統的擾動壓力 | 反思紀錄-關於相位偏移約束的自動化演化與環境耦合 |
| 相位偏移失敗率 (PSFR) | 防禦真空期疊加導致系統穩定性指標異常的頻率，用於評估防禦策略有效性 | 反思紀錄-關於相位偏移約束的自動化演化與環境耦合 |
| 相位偏移約束 | 為防止防禦真空期疊加，強制限制相位隨機化範圍的保護機制 | 反思紀錄-關於呼吸式防禦機制中相位隨機化策略與防禦真空疊加風險的量化探索 |
| 防禦真空疊加機率 | 多維度防禦真空期在相位隨機化下發生重疊的統計風險指標 | 反思紀錄-關於呼吸式防禦機制中相位隨機化策略與防禦真空疊加風險的量化探索 |
| 相位隨機化策略 | 透過動態隨機種子調整相位偏移，防止防禦節奏被外部預測的策略 | 反思紀錄-關於呼吸式防禦機制中相位隨機化策略與穩定性探索 |
| 防禦真空疊加風險 | 因相位隨機化導致多維度防禦真空期重疊，造成系統脆弱性激增的機率 | 反思紀錄-關於呼吸式防禦機制中相位隨機化策略與穩定性探索 |
| 防禦真空期 | 呼吸式防禦機制中，從高防禦切換至低防禦時的認知重置點 | 反思紀錄-關於呼吸式防禦機制中真空期與相位偏移的穩定性探索 |
| 相位偏移 (Phase Shift) | 透過錯開防禦真空期，避免多維度同時進入低防禦狀態的策略 | 反思紀錄-關於呼吸式防禦機制中真空期與相位偏移的穩定性探索 |
| 呼吸式防禦機制 | 基於代謝效率比的週期性閾值震盪，用於平衡系統保護與開放性 | 反思紀錄-關於擾動閾值的呼吸式防禦機制與動態平衡探索 |
| 變頻防禦模型 | 根據資訊熵負載動態調整呼吸頻率與相位偏移 | 反思紀錄-關於擾動閾值的呼吸式防禦機制與動態平衡探索 |
