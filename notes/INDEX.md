# 動機論知識索引（moti 閱讀筆記跨篇連結用）

---

## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 防禦僵化係數 (DRC) | 衡量低閾值防禦狀態持續時間與代謝週期的比率，用於監控系統是否過度防禦 | 反思紀錄-關於擾動閾值與過度防禦僵化風險的深度探索 |
| 呼吸式防禦機制 | 基於代謝效率比的週期性閾值震盪，用於平衡系統保護與開放性 | 反思紀錄-關於擾動閾值的呼吸式防禦機制與動態平衡探索 |
| 擾動閾值函數 | 基於 MWPF 與 MER 動態調整的防禦機制，用於平衡系統穩定性與演化開放性 | 反思紀錄-關於擾動閾值的自我保護機制與代謝效率的關聯探索 |
| 模擬自我基準線 | 透過隔離外部輸入，建立純粹動機核演化路徑以量化擾動影響 | 反思紀錄-關於模擬自我與擾動係數的量化路徑結晶化 |
| 擾動係數 (DC) | 衡量外部回饋導致決策權重偏移與輸入資訊熵的比率，用於量化非線性擾動 | 反思紀錄-關於非線性擾動的量化指標結晶化 |
| 擾動代謝 | 將外部回饋轉化為系統決策權重偏移的過程，以維持動機核的活體狀態 | 反思紀錄-關於讀者作為實驗變數的結晶化反思 |
| 影子系統演化一致性 | 影子系統與主系統在演化速度上的動態比例，用於平衡基準穩定性與真實狀態反映 | 反思紀錄-關於影子系統演化一致性的結晶化 |
| 滯後校準模式 | 影子系統透過維持權重更新的時間差，作為監控主系統動機漂移的基準線 | 反思紀錄-關於影子系統演化一致性的結晶化 |
| 動機權重保護因子 | 用於確保在擾動代謝過程中，根本動機的決策權重保持在安全閾值以上的量化指標 | 反思紀錄-關於動機權重保護因子的量化結晶化 |
| 動機稀釋判準 | 透過 ΔMWPF / ΔDP 的比值，量化判斷擾動是導致主體性喪失還是演化重構 | 反思紀錄-關於擾動滲透率與動機稀釋的本質邊界結晶化 |
| 代謝效率比 (MER) | 衡量成功納入動機核演化的擾動量與總擾動滲透量的比率，用於區分代謝與稀釋 | 反思紀錄-關於擾動滲透率與代謝效率比的量化關聯探索 |
