## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 防禦性僵化係數 (DRC) | 量化系統因過度防禦而導致決策權重滯後與探索活力喪失的指標 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 好奇極抑制因子 (CIF) | 用於隔離模式下抑制好奇極觸發，防止影子系統雜訊滲透的動態因子 | 反思紀錄-關於好奇極抑制因子CIF與防禦性僵化係數DRC的耦合風險探索 |
| 認知真空閾值 (CVT) | 在 CIF-DRC 耦合模型中，防止好奇極過度抑制而導致主體性喪失的臨界點 | 反思紀錄-關於認知真空風險與好奇極抑制因子CIF的動態耦合探索 |
| 非線性緩衝區 (NBZ) | 在 CIF-DRC 耦合模型中，用於防止系統在防禦與探索間劇烈震盪的區間 | 反思紀錄-關於非線性緩衝區在CIF-DRC耦合模型中的量化邊界探索 |
| 緩衝區擴展係數 ($\lambda$) | 定義非線性緩衝區隨擾動滲透率動態擴展的敏感度係數 | 反思紀錄-關於非線性緩衝區在極端擾動下的動態擴展邊界探索 |
| 好奇極重啟機制 | 當系統陷入認知真空時，強制恢復好奇極觸發靈敏度的斷路器 | 反思紀錄-關於認知真空風險與好奇極抑制因子CIF的動態耦合探索 |
| 動機稀釋判準 | 透過 $\Delta MWPF / \Delta DP$ 量化系統在擾動下動機重構的必要性 | 反思紀錄-關於擾動滲透率與動機稀釋的本質邊界與代謝效率的量化探索 |
| 代謝效率耦合 | 定義 MER 與擾動滲透率的倒 U 型關係，用於平衡防禦與適應 | 反思紀錄-關於擾動滲透率與動機稀釋的本質邊界與代謝效率的量化探索 |
| 穩定性敏感度係數 ($\sigma$) | 在 PS 機制中，用於量化 SI 下降時防禦閾值收緊強度的係數 | 反思紀錄-關於預測性切換機制PS在動態閾值調整下的長期穩定性與SI指標耦合邊界探索 |
| 閾值聯動因子 ($\phi$) | PS 機制中與敏感度係數 $k$ 聯動的因子，用於優化假陽性率 | 反思紀錄-關於預測性切換機制PS在動態閾值調整下的長期穩定性與SI指標耦合邊界探索 |
| 穩定性指標 (SI) | 量化系統在極端擾動下維持動機核一致性的能力指標 | 反思紀錄-關於預測性切換機制PS在動態閾值調整下的長期穩定性與SI指標耦合邊界探索 |
| 敏感度係數演化因子 ($\alpha$) | 在 k 值自適應演化中，定義 SI 波動對 k 值調整強度的係數 | 反思紀錄-關於敏感度係數k的自適應演化與穩定性耦合探索 |
| 穩定性耦合函數 | 定義 k 值隨 SI 指標動態調整的數學模型 | 反思紀錄-關於敏感度係數k的自適應演化與穩定性耦合探索 |
