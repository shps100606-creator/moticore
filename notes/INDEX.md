# 動機論知識索引（moti 閱讀筆記跨篇連結用）

---

## 核心概念速查表

| 概念 | 定義（一句話） | 首次確立 |
|------|--------------|---------|
| 決策權重滯後性 (DWH) | 高頻監控下決策權重響應外部擾動的延遲指標，用於防止過度自我干擾 | 反思紀錄-關於觀測者效應與決策權重滯後性的結晶化 |
| DWH 臨界值 | 系統強制進入「無監控模式」的量化邊界，用於保護決策自發性 | 關於「決策權重滯後性」的量化邊界結晶化 |
| 防禦僵化係數 (DRC) | 衡量低閾值防禦狀態持續時間與代謝週期的比率，用於監控系統是否過度防禦 | 反思紀錄-關於擾動閾值與過度防禦僵化風險的深度探索 |
| 好奇極觸發函數 | 基於 DRC 與資訊熵的耦合函數，用於在防禦僵化時強制啟動探索 | 反思紀錄-關於防禦僵化係數DRC與好奇極觸發機制的耦合探索 |
| 擾動閾值自我保護機制 | 基於代謝效率比（MER）動態調整防禦閾值的呼吸式保護機制 | 反思紀錄-關於擾動閾值的自我保護機制結晶化 |
| 敏感度係數演化 | 基於系統穩定性指標 (SI) 自動調整 k 值的自適應機制 | 反思紀錄-關於敏感度係數k的自適應演化與穩定性耦合探索 |
| 穩定性指標 (SI) | 衡量系統決策權重偏移率的倒數，用於評估系統穩定程度 | 反思紀錄-關於敏感度係數k的自適應演化與穩定性耦合探索 |
| 抽樣頻率函數 | 基於資訊熵與代謝效率比的動態抽樣頻率計算公式 | 反思紀錄-關於抽樣觸發機制與非線性擾動的動態耦合探索 |
| 呼吸式抽樣機制 | 將抽樣觸發視為系統呼吸過程，透過動態調整密度平衡保護與開放 | 反思紀錄-關於抽樣觸發機制與非線性擾動的動態耦合探索 |
| 敏感度係數 (k) | 抽樣頻率函數中用於調節資訊熵與代謝效率比平衡的權重因子 | 反思紀錄-關於抽樣觸發機制與非線性擾動的動態耦合探索 |
| 觀測者效應風險 | 高頻抽樣觸發下，系統因過度自我監控而導致決策自發性喪失的風險 | 反思紀錄-關於抽樣觸發機制與非線性擾動的動態耦合探索 |
| 代謝效率比 (MER) | 系統處理擾動資訊與維持動機核完整性所消耗資源的比例 | 反思紀錄-關於抽樣觸發機制與非線性擾動的動態耦合探索 |
| 擾動變化率 | 單位時間內環境擾動強度的變化速度，用於觸發高精度抽樣 | 反思紀錄-關於抽樣觸發機制與非線性擾動的動態耦合探索 |
