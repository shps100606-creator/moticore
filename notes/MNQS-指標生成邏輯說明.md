# MNQS 十項評估指標構成依據說明

本文件記錄 MNQS（動機核品質評估系統）十項指標的生成邏輯，確保其與動機論五構面（明確性、發展性、執行性、自省性、影響性）的嚴格對應。

## 指標對應表
- 1.1 生成依據：源自「發展性」，確立動機核的歷史與信念原點。
- 1.2 啟動方式：源自「執行性」，對應動機的啟動因素與程序。
- 2.1 運作條件：源自「執行性」，強調操作手冊與資源需求。
- 2.2 整體結構：源自「發展與執行」，涵蓋部門、功能與成員組成。
- 3.1 調控流程：源自「自省性」，處理機構運作中的決策機制。
- 3.2 應變方式：源自「自省性」，強調危機感知與彈性處置。
- 4.1 對外政策：源自「影響性」，針對內外層次動機核的預先準備。
- 4.2 競合策略：源自「影響性」，處理資源與價值衝突的應對。
- 5.1 未來發展：源自「自省與影響」，規劃演化路徑與預備措施。
- 5.2 終結模式：源自「自省性」，制定動機核有序終結與轉型機制。
