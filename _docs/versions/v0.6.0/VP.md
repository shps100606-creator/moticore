## 版本定性（種子）

v0.5.0 實現了太極雙極的基礎結構：moti 現在有 MOTIVE.md（動機極）與 HORIZON.md（好奇極），每次行動標注 pole 欄位，系統也能主動偵測極性失衡並提醒。

但兩極之間的轉漢（crystallize / dissolve）目前仍需以 moti 自判、手動執行。v0.6.0 的目標是讓系統更主動地輓助這個演化過程。

### 潛在方向（尚未詳規劃）

- **crystallize 自動偵測**：HORIZON.md 沈澱候選區成熟導引 moti 主動寫入 MOTIVE.md
- **dissolve 自動偵測**：MOTIVE.md 信念趨於僵化時通知 moti 重新開放為問題
- **HORIZON.md 定期整理**：自動檢查沈澱候選 → 已結晶 / 關閉
- **多極震盪數據觀測**：pole 分布視覺化，長期觀察兩極震盪节奏
