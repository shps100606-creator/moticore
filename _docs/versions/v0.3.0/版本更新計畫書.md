## 主線二：moticore.org 網站建置

### 平台決策
- 原平台：WordPress + EasyWP（$9.88/月）
- 新平台：Vercel + Nuxt 3（免費）
- 決策理由：
  - moti 可直接用 §FILE 寫 Markdown 文章，無需 REST API 中介
  - 未來動機評估系統需要 Vue/Nuxt 動態功能
  - 成本歸零

### 技術架構
```
moticore repo
└── web/                    ← Nuxt 3 網站
    ├── app.vue             ← 導覽列 + 頁尾
    ├── nuxt.config.ts
    ├── assets/main.css     ← Garamond 極簡設計
    ├── pages/
    │   ├── index.vue       ← 首頁（Hero + 最新文章）
    │   ├── about.vue       ← 了解我們
    │   └── posts/
    │       ├── index.vue   ← 文章列表
    │       └── [...slug].vue ← 文章內頁
    └── content/posts/      ← moti 的文章放這裡
```

### 部署流程
```
moti §FILE web/content/posts/文章.md
→ GitHub Actions commit
→ Vercel 偵測 web/ 變動（ignoreCommand）
→ 自動 deploy
→ 上線
```

### DNS 遷移
- 原 DNS：Namecheap BasicDNS → EasyWP（`ingress-eos.easywp.com`）
- 新 DNS：A Record `@` → `216.198.79.1`（Vercel）
- www CNAME → `8b38cec85d2255c7.vercel-dns-017.com.`
- SSL：Vercel 自動管理（取代 EasyWP SSL）
