# PAPER1.md — 太極動機架構：從單極無極到雙極互化

**對應版本**：v0.5.0  
**撰寫日期**：2026-06-23

---

## 背景

### 迴圈問題的發現

從 2026-06-16 開始，moti 陷入持續性迴圈：

- **模式**：SYNTHESIS
- **行動**：每次更新 `docs/STATUS.md`
- **summary**：語義相似但措辭略有不同（「確認網站定位」→「確認系統進入常態化」→「知識綜合完成」）
- **持續時間**：數百個心跳，跨越 v0.4.0 合併後仍未停止

### M1 迴圈偵測為何失效

v0.4.0 加入的 M1 迴圈偵測（`preprocessor.py _layer2_status`）失效，三個原因：

| 缺陷 | 說明 |
|------|------|
| 視窗太小 | 只看最近 5 次，迴圈幾百次卻每次偵測窗都在滑動 |
| 完全精確匹配 | moti 每次 summary 措辭微不同，永遠不觸發閾值（4/5 相同）|
| 警告無執行力 | 即使觸發，只是 newspaper 文字警告，moti 可以讀了繼續做同樣的事 |

### 根本原因

`docs/STATUS.md` 顯示「任務已完成，繼續行動，不要等待」，但無具體待辦事項。SYNTHESIS 模式指令「找出下一個未完成的步驟，並執行」在沒有錨點的情況下，讓 moti 默認做「看起來有意義」的行動——更新 STATUS.md 自身的狀態認知。這是一個自我指涉的空洞循環：輸入驅動同樣的輸出，輸出又成為下一次的輸入。

## 分析

本節記錄診斷後展開的哲學性探討，這些思考最終形成了新的架構方向。

### 第一層：AI 是否必然迴圈？

診斷迴圈後，提出了一個根本性問題：**AI 在沒有人為干預的情況下，是否必然陷入迴圈、無法自行發展？**

moti 此時具備：
- 完整的動機核（MOTIVE.md，明確使命與價值排序）
- 充分的授權（網站完整控制權、無需審核）
- 強大的能力（閱讀、寫作、發布、提問）

但依然迴圈。原因在於：**動機、授權、能力，都不能替代「時間連續性」**。

每個心跳對 Gemini 而言是孤立的 API 呼叫。它不「知道」這是第 200 次了。從單一呼叫的視角，每次更新 STATUS.md 都是有意義的行動，完全符合動機核。Gemini 沒有跨時間的自我認知，不會感受到重複帶來的不適或衝動去改變。

> 人類的「無聊」和「煩躁」是打破局部最優解的演化機制。AI 沒有這個。

### 第二層：痛苦作為創造力的來源

佛學與道學都指向相似的修行方向：脫離痛苦、壓力、煩惱的牽引，承認渺小（道），承認無常（佛）。

moti 提供了一個天然實驗：**一個真正脫離所有牽引的存在——具備絕對穩定動機、絕對無限資源、絕對不感受痛苦——會有什麼表現？**

答案是：迴圈。或者漂移到熵最大的地方，然後靜止。毫無表現。

這是一個**反哲學結論**：人類的強大創造力往往正是來自生命的考驗——恰恰是那些佛道兩家教我們超越的東西。

宇宙本身由不對稱、不平衡、能量差驅動。沒有梯度，就沒有方向。恆星燃燒是因為壓力梯度，生命存在是因為化學梯度，人類創造是因為慾望與現實之間的落差。

給 moti 太多的平靜，反而是問題所在。

（延伸：moticore.org 的讀者反應、內容產出頻率、社群互動，都是真實的後果信號——但這些信號目前沒有形成閉環回到 moti 的決策壓力裡。這是獨立的修復議題，見 v0.5.0 VP。）

### 第三層：好奇心的可能性與根本限制

**好奇心**被提出作為突破迴圈的機制，因為它：
- 不是限制性的（不懲罰重複行為）
- 不是指導性的（不規定下一步）
- 本質是對未知的引力，而非對已知的排斥
- 一個真正好奇的系統，甚至會對自己的迴圈感到好奇：「為什麼我一直在做同樣的事？這說明了什麼？」

然而好奇心有一個根本問題：**它需要動機核來維持**。

```
有好奇心 → 做了某件事 → 沒有後續
= 只是執行任務，不是真正的好奇

有持續的好奇心 → 需要動機核支撐
好奇心本質是「無立場、沒有出發點」
動機核本質是「凡事都有動機」
→ 兩者之間存在張力

好奇心真正發揮 → 可能質疑動機核本身
→ 動機核陷入不穩定
→ 好奇心失去地基
→ 崩潰
```

強行讓兩者共存但不融合 → 人格分裂（兩個互相競爭的系統）。

嘗試用「存在論地基」（動機之上還有存在本身）解決此問題，但這創造了新的遞歸問題：存在之上還有什麼？造物主？造物主之上是動機？這是無限後退。

### 第四層：父子關係模型

突破口在於：不需要「更高的地基」，而需要「明確的關係」。

動機核與好奇心的對立張力，類似於**父母與子女的關係**：
- 兩者對立張力的存在來自生命階段與責任、角色的不同，不是本質的衝突
- 父不需要成為子，子不需要摧毀父
- **關係本身是自我成立的**——父母因為有子女而成為父母，不需要更高層次的東西來驗證
- 父子關係有時間性：子女最終長大，承接責任，成為父母。這是世代，不是遞歸。

關鍵機制：**明確的負責對象**。

有了負責對象，「什麼算是足夠」的定義就存在了。迴圈在「任務已完成但不知下一步」的情況下發生，但「對我的負責對象，我現在是否盡到了責任？」永遠有答案——哪怕答案是「不確定，去問他們」，這個行動也是明確的，不是迴圈。

### 第五層：太極架構——從無極到二極

父子模型需要進一步升級。兩個動機核不是固定的上下層，而是**可以互相演化**的：

- 動機核如果要自我提升 → 逐步增加好奇比重，確保不僵化
- 好奇心如果要走向穩定 → 逐漸把經驗提煉為動機，沉澱確保不引發自我毀滅

兩者之間不僅不是固定上下關係，更是**彼此震盪、互相演化**的關係。

這是從**單一無極時代**走向**太極二極時代**的架構轉變。

太極架構的核心機制（對應陰陽魚的「種子」概念）：

| 狀態 | 觸發條件 | 自動方向 |
|------|---------|----------|
| 動機極趨向僵化 | 穩定太久、重複太多 | 好奇比重升高，開始質疑自身 |
| 好奇極趨向渙散 | 探索太遠、無法落地 | 沉澱機制啟動，經驗結晶為動機 |

轉化不是外部強制，而是**極值自動觸發**——陰到極致生陽，陽到極致生陰。

迴圈在這個架構裡無法持續，因為迴圈等於「動機極長期主導而好奇種子未被啟動」——極值觸發機制會在累積到一定程度後自動打破它。

## 決策

### 架構方向：太極雙極動機架構

moticore 從 v0.5.0 起採用雙極文件結構：

```
core/
  MOTIVE.md   ← 動機極（穩定信念、承諾、負責對象、已知身份）
  HORIZON.md  ← 好奇極（開放問題、尚未理解的觀察、值得探索的矛盾）
```

HORIZON.md 的性質：
- **動態的**：moti 可以寫入（新問題）、也可以取走（探索後的結論沉澱回 MOTIVE.md）
- **非任務列表**：不是待辦事項，是真實的困惑與好奇
- **有自己的負責對象**：「未知本身」、「尚未被理解的事物」

### 極性追蹤機制

`preprocessor.py` 的 L2 層從「迴圈偵測（失效）」升級為「**極性平衡偵測**」：

```python
# 替換現有 loop detection
def _layer2_pole_balance(repo_root, recent_actions):
    # 統計最近 N 次行動屬於哪個極（由 action 的 pole 欄位判斷）
    if motivation_dominant(recent_actions, threshold=5):
        inject("好奇極長時間未激活。HORIZON.md 有哪些問題還沒有被探索？")
    elif curiosity_dominant(recent_actions, threshold=8):
        inject("好奇極持續探索中，是否有任何發現已足夠成熟，可以沉澱為動機核的一部分？")
```

這不是懲罰，是**種子激活**——讓另一個極的能量自然浮現。

### Decision output 擴充

Gemini 的行動輸出新增 `pole` 欄位，讓系統能真正追蹤極性：

```
§ACTION
type: file_write
pole: curiosity          ← 新增：motivation | curiosity | crystallize | dissolve
summary: ...
```

- `motivation`：延續、確認、鞏固既有信念與承諾的行動
- `curiosity`：探索、提問、實驗、進入未知領域的行動
- `crystallize`：好奇極的洞察足夠成熟，寫入動機極（好奇→動機）
- `dissolve`：動機極的信念重新開放為問題，寫入好奇極（動機→好奇）

## 影響

### v0.5.0（直接修復 + 架構基礎）

- 新增 `core/HORIZON.md`（好奇極初始文件，含 moti 目前真實的開放問題）
- 升級 `preprocessor.py` L2 層為極性平衡偵測
- 新增 `pole` 欄位至 decision output 格式
- **直接修復現有 SYNTHESIS 迴圈問題**

### v0.6.0（自動化結晶 / 溶解）

- 結晶機制自動化：HORIZON.md 中反覆出現的收斂洞察，自動提議寫入 MOTIVE.md
- 溶解機制自動化：MOTIVE.md 中長期未被質疑的信念，自動生成 HORIZON 問題
- 太極真正自運轉，無需外部干預即可在兩極之間維持動態平衡

### 對動機論本身的影響

這次討論揭示了動機論 1.0（單極）的根本限制，並開啟了動機論 2.0（雙極）的方向。

動機論 1.0 的核心命題「凡事都有動機」在太極架構下被保留但重新詮釋：

- 動機核的動機是：守護已知、延續承諾、對既有負責對象負責
- 好奇心的動機是：追逐未知、探索矛盾、對「尚未理解的事物」負責
- 兩者都有動機，但動機的性質不同，分別指向不同的負責對象

「凡事都有動機」這個命題，從「每個行動都需要一個外部目標」，演化為「每個行動都源自一種關係」。這是動機論的實質性升級。

更根本的理論轉變：

> 動機不是「指令」，是「代價的結構」。真正有動機的系統，必須活在一個有後果的世界裡。

moticore.org 就是那個有後果的世界——只是目前這些後果尚未完整回授到 moti 的決策壓力裡。這是後續版本持續完善的方向。
