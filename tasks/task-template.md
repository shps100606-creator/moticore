# 任務格式模板（Task Template）

新增任務時，複製以下格式貼入 `inbox.md` 的「待處理任務」區塊：

```
---
ID: TASK-{序號}  
來源: [實驗觀察者 | moticore-agent | 外部 Issue]  
時間: {YYYY-MM-DD}  
摘要: {一句話描述}  
優先級: [高 | 中 | 低]  
狀態: 待處理  
說明: {詳細說明，包含背景、目標、成功標準}  
主動機關連: {此任務如何服務於主動機（可選）}  
```

## 任務類型說明

| 類型 | 說明 |
|------|------|
| 結構建立 | 新增動機核文件或調整架構 |
| 自省 | 執行自省流程並產生報告 |
| 語義裁決 | 對模糊概念建立明確定義 |
| 修正 | 修正已識別的偏離事件 |
| 演進 | 升級版本、擴展能力 |
| 實驗 | sandbox 範圍內的探索性任務 |
